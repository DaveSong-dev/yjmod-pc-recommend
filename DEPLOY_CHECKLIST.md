# 최종 배포 전 점검 체크리스트

## 1) 데이터/크롤링
- [ ] `data/pc_data.json`이 최신이며 `in_stock=false` 데이터가 정상 제외되는지 확인
- [ ] `last_updated` 값이 갱신되고, 화면의 "몇 분 전" 표시가 정상 동작하는지 확인
- [ ] 24/36개월 무이자 상품의 `price_display`가 월 납부금 기준으로 노출되는지 확인
- [ ] 카테고리(게이밍/사무/AI/영상편집/3D/방송/24/36개월) 분류 누락 여부 확인

## 2) UI/UX 핵심 시나리오
- [ ] 기본 진입 시 그룹별 카드가 즉시 보이는지 확인 (카드 숨김 상태 없음)
- [ ] 상단 그룹 `더보기` 클릭 시 필터 전환 + 결과 노출이 정상인지 확인
- [ ] 하단 `N개 더 보기` 클릭 시 버튼 비활성 + 로딩 스켈레톤 + 추가 카드 노출 확인
- [ ] 필터 조합 후 `더보기` 반복 클릭 시 중복 클릭/이중 로딩이 없는지 확인
- [ ] 위자드 색상 필터(화이트/블랙/RGB) 결과가 의도대로 좁혀지는지 확인

## 3) 성능
- [ ] 카드 이미지 `loading="lazy"` + `decoding="async"`가 적용되어 있는지 확인
- [ ] 긴 목록에서 스크롤 시 프레임 드랍/먹통 현상이 없는지 확인
- [ ] 모바일에서 과도한 레이아웃 시프트(CLS) 없이 카드가 안정적으로 렌더되는지 확인
- [ ] 첫 로드 시 불필요한 콘솔 에러/경고가 없는지 확인

## 4) 브라우저/해상도 검증
- [ ] Chrome 최신 버전 (Windows)에서 주요 플로우 정상
- [ ] 모바일 폭(360px~430px)에서 필터 바/카드/버튼 터치 동작 정상
- [ ] 태블릿/데스크톱 해상도에서 카드 그리드 열 수/간격이 깨지지 않는지 확인

## 5) 배포 직전 운영 체크
- [ ] 실제 운영 URL로 링크(공식 홈페이지/네이버 카페/인스타/상담)가 올바른지 확인
- [ ] `favicon.ico` 준비 (404 방지)
- [ ] 캐시 이슈가 있으면 배포 후 강력 새로고침 안내 문구 준비
- [ ] 롤백용 이전 정적 파일 백업 확보

## 6) 배포 후 스모크 테스트
- [ ] 첫 진입 후 10초 이내 메인 카드 확인 가능
- [ ] 필터 1개 적용 + 더보기 2회 + 초기화까지 1회 시나리오 통과
- [ ] 위자드 3단계 완료 후 추천 카드 노출 확인
- [ ] 실시간 출고 현황 카드 링크 이동 확인
